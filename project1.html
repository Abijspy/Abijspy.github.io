Chatbot-Using-Python

This Project is based on Tkinter using python for GUI and AI BOT using JSON file for AI.

we proposed an system which will work as an application and give users information about present health. The system finds out what the user actually wants by retrieving the semantic meaning of the query. Then the system will pass on this semantic text as input to the pattern matching algorithm. The patching algorithm make user of knowledge that is already fed into it in order to generate a response.

As we can see in the images or code output is showing the clear information of Severe Symptoms. And we can see those things in Two Formats.One is using GUI with Python and the another is using JSON file with Python. Both are showing the Python code of chatbot for Corona Virus.
